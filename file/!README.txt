Что, за что отвечает:
main - Отвечает за середину файла, когда выбрана главная страница + нижние последние 10 подключенных вендера.
mod_leftmenu - Выводит список всех категорий в левой менюшке
index - Там есть карусель для вендеров(сверху справа), а также вывод всей разметки HTML
mod_partner - Вывод партнеров справа на сайте(НАШИ ПАРТНЕРЫ)
vender - Страница вендеров с информацией о них, также кнопка уведомление.
product - Отвечает за вывод информации о продукте
mod_topmenu - Отвечает за желтую полосу "Где получить скидки" и т.д.
mod_enter - Когда входит пользователь, туда записываются все сессии
reg - Моя регистрация
shopcart - Покупка и обработка подарков
myspace - Отвечает за личный кабинет
hitua - Баннер с выводом информации о кол-ве посетителей
mail - Когда отсылается почта через меню "Контакты"
